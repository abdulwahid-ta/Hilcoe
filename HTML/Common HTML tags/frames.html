Deprecated Example: Frameset

The <frameset> and <frame> elements were used in older HTML versions to divide the browser window into sections. They are **deprecated** and should no longer be used, but are shown here for reference. 

<frameset cols="50%,50%">
  <frame src="index.html">
  <frame src="about.html">
</frameset>
